# 📊 Data Technician Workbook – Week 1 (Excel Project)

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp (Week 1)** and focuses on analysing **retail and sales data using Microsoft Excel (2026)**. The workbook demonstrates core data handling, transformation, and analysis skills using built-in Excel functions and tools.

The aim of the project was to clean, organise, and extract meaningful insights from raw sales data through formulas, sorting, filtering, and visualisation.

---

## 🛠️ Skills & Techniques Used

### 🧮 Excel Formulas & Functions
- **Basic Aggregations**
  - `SUM` – calculate total sales
  - `AVERAGE` – determine average sales values
- **Conditional Calculations**
  - `SUMIF` – sum sales based on specific criteria (e.g. region or category)
  - `AVERAGEIF` – calculate averages based on conditions
- **Date Functions**
  - `DATE` – construct structured date values
  - `MONTH` – extract month from date for trend analysis
  - `YEAR` – analyse yearly performance
- **Lookup & Categorisation**
  - `VLOOKUP` – retrieve product/category data from reference tables
  - `SWITCH` – map values into readable categories
- **Dynamic Data Functions**
  - `UNIQUE` – extract distinct values (e.g. customer segments, product names)

---

### 🔍 Data Preparation & Exploration
- Data **cleaning and structuring**
- **Sorting** (ascending/descending) to identify top-performing products or regions
- **Filtering** to isolate specific data subsets (e.g. monthly sales, regions)

---

### 📊 Data Analysis
- Creation of **PivotTables** to:
  - Summarise sales by category, region, and time period
  - Compare performance across different dimensions
- Identification of trends and patterns in retail data

---

### 📈 Data Visualisation
- Built clear and insightful **charts**, including:
  - Bar charts (sales by category)
  - Line charts (monthly trends)
  - Pie charts (distribution of sales)
- Used visuals to communicate key business insights effectively

## 📂 Project Structure
